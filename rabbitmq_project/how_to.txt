Как запускать это всё после перезапуска (шаги):

ШАГ 1: Запускаешь RabbitMQ (главная очередь):
docker run -d --rm --name my_rabbit -p 5672:5672 -p 15672:15672 rabbitmq:3-management

ШАГ 2: Запускаешь рабочих (хранителей):
Правой кнопкой по файлу `worker.py` -> `Run 'worker'`
Правой кнопкой по файлу `worker.py` -> `Run 'worker'` (еще раз, для второго рабочего)
И так столько раз, сколько рабочих хочешь (например, 3 раза).

ШАГ 3: Запускаешь менеджера (главного):
Правой кнопкой по файлу `manager.py` -> `Run 'manager'`
Он запустится и скажет что-то типа "Жду команд".

ШАГ 4: Запускаешь клиент (консоль, с которой ты будешь говорить):
Правой кнопкой по файлу `client.py` -> `Run 'client'`
Появится приглашение для ввода команд `> `.

ШАГ 5: Работаешь!** В консоли клиента вводишь команды:
start` - просто проверяем что все живы
`load /путь/к/папке e` - скачать и раскидать файлы равномерно
`find hello` - найти все предложения со словом "hello"
`purge` - очистить всё
`exit` - выйти
